haloCaps ⌨

让 Windows 像 macOS 一样丝滑 ✨

支持把 CapsLock 和其他键组合，实现光标移动、删除等常用操作，告别手指长距离跨越，大幅提升编辑效率

⬇️ 下载 Releases ⭐ GitHub 仓库 ⚙️ 配置示例

✅ 目前支持的键位

  • QWER 组合：Caps+Q/W/E/R
  • ASDF 组合：Caps+A/S/D/F
  • 共 8 个键位，可完全自定义功能

🎯 核心功能（类 macOS 快捷键）

  • ↑ 上移光标：keyFunc_moveUp
  • ↓ 下移光标：keyFunc_moveDown
  • ← 左移光标：keyFunc_moveLeft
  • → 右移光标：keyFunc_moveRight
  • ⌫ 删除字符：keyFunc_delete
  • ↩ 回车确认：keyFunc_enter

⚙️ 高级定制

  • 自定义快捷键映射：[remap]
  • 例：!c 映射为 ^c
  • 支持 Alt、Ctrl、Shift、Win 组合
  • 编辑 conf.ini 即可生效

🚀 本地快捷键

  • 按住 Caps 500ms 内单击键位触发
  • 支持 Alt+C、Alt+V 等系统快捷键重映
  • 轻量级：仅 2.8MB，内存占用极小
  • 开机自启：配置 autostart=true
